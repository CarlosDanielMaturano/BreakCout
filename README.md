# <span color>Break<i style="color: #5C6BC0;">C</i>out</span>
Breakout, but made with C

### Dependencies: 
- make
- raylib
- clang

### Building the project:
```bash
make build
```

### Running the project:
```bash
make run
```

### Building & Running project:
```bash
make all
```
